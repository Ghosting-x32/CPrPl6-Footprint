从标准输入流读取数据的程序一般使用while 循环，其形式是：
char ch;
while(cin >> ch)
{
/* ... */
}
因为这项功能事实上不太要求我们严格跟踪循环变量的变化过程，所以改写成
for 循环后稍显冗余：
for( ; cin >> ch; )
{
/* ... */
}
请注意：在上面的for 循环控制结构中，第一条语句和第三条语句都是空语句，
额外添加一个循环控制变量（比如int i）是没有意义的。
整数累加求和的程序一般使用for 循环，其形式是：
int iCount = 0;
for(int i = 0; i < 10; ++i)
{
iCount += i;
}
使用while 循环改写后的形式是：
int iCount = 0, i = 0;
while(i < 10)
{
iCount += i;
++i;
}
在大多数情况下，两种循环形式可以互相转换。如果只能使用一种循环，作者
倾向于使用for 循环。for 循环的优点是结构严谨，便于控制程序的逻辑。
