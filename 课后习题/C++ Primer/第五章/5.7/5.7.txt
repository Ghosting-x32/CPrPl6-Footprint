if (ival1 != ival2)

(b) if 语句的循环体只能是一条语句，本题从逻辑上来说需要做两件事，一是修
改minval 的值，二是重置occurs 的值，所以必须把这两条语句放在一个块里。
程序修改为：
if (ival < minval)
{
minval = ival;
occurs = 1;
}
